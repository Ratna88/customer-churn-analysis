# Customer Churn Analysis 📊

## 📌 Project Overview

This project analyzes customer churn data using Python and Power BI to identify patterns and factors affecting customer retention.

## 🛠️ Tools Used

* Python (Pandas, Matplotlib, Seaborn)
* Google Colab
* Power BI

## 🔍 Key Steps

* Data Cleaning & Preprocessing
* Feature Encoding
* Data Visualization
* Dashboard Creation in Power BI

## 📈 Key Insights

* Customers with month-to-month contracts churn more
* Higher monthly charges increase churn probability
* Certain payment methods show higher churn rates

## 📂 Files Included

* `final_cleaned_data.csv` → Cleaned dataset
* `churn_analysis.ipynb` → Data cleaning & analysis
* `dashboard.pbix` → Power BI dashboard

## 🚀 Outcome

Built an end-to-end data analysis project suitable for data analyst roles.
